# 今日打卡了吗？

一款简洁优雅的学习打卡应用，帮助您养成每日学习的好习惯。

## 功能特性

### 📅 每日打卡
- 一键打卡，记录每天的学习成果
- 打卡状态可视化，日历上清晰标记已打卡/未打卡日期
- 打卡统计：完成率、已打卡天数一目了然

### 📆 日历视图
- 月历展示，左右滑动切换月份
- 日期选择器快速跳转到指定月份
- 点击日期查看或创建当日日程

### 📝 日程管理
- 为任意日期添加学习日程
- 设置日程时间和提醒
- 日程列表展示，方便管理

### 💡 每日名言
- 每日自动获取励志名言
- 点击可刷新获取新名言
- 网络异常时自动使用本地备用名言

### 🎨 精美界面
- 根据星期几自动切换主题配色
- 流畅的动画效果
- 现代化的卡片式设计

## 技术栈

- **前端框架**: uni-app (Vue 3)
- **云服务**: uniCloud (阿里云)
- **数据存储**: 本地存储 (localStorage)

## 开发环境

### 前置要求

- [HBuilderX](https://www.dcloud.io/hbuilderx.html) (推荐使用最新版本)
- Node.js (可选，用于部分插件)

### 运行项目

1. 使用 HBuilderX 打开项目目录
2. 点击菜单栏 `运行` -> 选择运行平台（浏览器/模拟器/真机）
3. 项目将自动编译并运行

### 打包发布

1. 点击菜单栏 `发行` -> 选择打包方式
2. 配置应用签名和证书
3. 等待打包完成，下载安装包

## 数据说明

应用使用本地存储保存以下数据：

| 存储键 | 说明 |
|--------|------|
| `checkInData` | 打卡记录数组 |
| `scheduleData` | 日程数据数组 |
| `dailyQuoteStateV2` | 每日名言缓存 |

## API 说明

应用使用 [一言 API](https://hitokoto.cn/) 获取每日名言：

- 接口地址: `https://v1.hitokoto.cn/`
- 请求方式: GET
- 超时时间: 5秒
